# Function introduction

## Ride Usage <a id="&#x5750;&#x9A91;&#x7684;&#x4F7F;&#x7528;"></a>

* Right-click the pet \(No items in main hand\)
* Ride pet following command

`/imipet ride`

* Ride the pet in designated slot ：

`/imipet ride pack_slot(1-6)`

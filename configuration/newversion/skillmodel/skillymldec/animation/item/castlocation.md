# 释放位置

```text
skill:
  animation:
    动态ID:
      type: item
      item:
        castLocation: "值"
```

### 

### 在释放者的位置基础上增加的值

```text
addLocation:方向类型:乘数:高度
```



### 跟随释放者

```text
follow:方向类型:乘数:高度
```



### 方向类型

```text
方向类型可选 left right behind
```



### 乘数

```text
乘数可正可负，越大距离越远
```



### 高度

```text
高度可正可负，无需解释
```
